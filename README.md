AI Workflow Explanation:

In our system, the workflow start with user entering prompts through the "Chatbot" tab located at the navigation bar at the top of page, with options of local AI model and Gemini API provided at user's choice. 

On clicking the search button, the user's message/prompt is sent to the backend with POST method. Depend on the model chosen by the user, the message is sent to the local embedding model "sentence-transformers/all-MiniLM-L6-v2" and LLM "vhab10/llama-3-8b-merged-linear" for process or to Google Gemini "gemini-3.1-flash-lite-preview".

Either way, the output is generated by the models with pre-built prompts and data passed from the apartments listings database, asking the model to rank the data and take the top 3 listings parsed into JSON format.

Then it is sent to the website for display.
